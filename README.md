# ClearMenu

Small QOL mod for Celeste

## What It Does
- Hides main menu buttons except `Climb`
- Adds main menu keybinds
    - `1` opens `Options`
    - `2` opens `Mod Options`
- Hides `Confirm / Back` keeping other mod prompts visible

## Build Script
- Take the `.csproj` inside `Source/`
- Builds the dll in `bin/`
- Output is `ClearMenu.zip` in the mod root
